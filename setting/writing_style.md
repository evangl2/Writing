# 写作风格指南：叙述语感锚定

**写完每段，问自己：这段话像不像从下面某本书里翻出来的？不像，重写。**

本书的叙述语感锚定以下四部真实中文MG作品。遇到拿不准的写法，回来对照原文。

---

## 一、锚定作品与核心语感

### 1. 任溶溶译《查理和巧克力工厂》（罗尔德·达尔）

**原文参照：**

> 查理·巴克特一进这间巨大的房间便瞪大眼睛四下打量起来。这儿真象是一间巫婆的厨房！四周全是各种黑色的金属罐子，这些罐子正在巨大的电炉上"噗噗"沸滚着，还有嘘嘘冒汽的汽锅、烧得嵫嵫作响的平底锅，以及发出哐啷哐啷、劈劈啪啪响声的奇怪的铁机器，天花板和墙上布满了管子，整个房间弥漫着烟气、水汽和浓郁芳香的气味。
>
> 而这时旺卡先生变得更加兴奋……他不停地一会儿跳到有柄平底锅边，一会儿又奔到机器旁，那副模样就象一个孩子围绕着他的圣诞礼物兴奋得手舞足蹈，简直不知道先看什么礼物好。他掀开一只大罐予的盖子嗅嗅，又窜过去把手指探进一大桶黏稠的黄色液体中去，然后拿出手指舔舔，尝尝味道……

---

### 2. 任溶溶译《了不起的狐狸爸爸》（罗尔德·达尔）

**原文参照：**

> "真痛呀！"狐狸先生说。
> "我知道你痛，亲爱的，但是很快就会好起来的。"
> "而且他很快就会长出来的，爸爸。"其中一个小狐狸说道。
> "他在也长不出来了，"狐狸先生说，"我下半生不会再有尾巴了。"
>
> ……"那是什么动静？"他敏捷的转过头去，倾听着。他现在听到的声音是一只狐狸所能听到的最可怕的声音——铁锨挖进土里的嚓嚓声。

---

### 3. 马爱农译《哈利·波特》（J.K.罗琳）

**原文参照：**

> 马车穿过两边有带翅野猪雕塑的大门，顺着宽敞的车道行驶着，由于狂风大作，马车剧烈地摇晃着。哈利靠在车窗上，看见霍格沃茨越来越近了，许多亮灯的窗户在厚厚的雨帘后面模模糊糊地闪着光。
>
> ……一个装满水的大红气球从天花板上落下来，在罗恩的头顶上爆炸了。罗恩浑身被浇得透湿，嘴巴里嘟嘟囔囔，跌跌撞撞地一闪，倒在旁边的哈利身上。就在这时，第二个水炸弹又落了下来，差一点儿击中赫敏，在哈利脚边爆炸了。冰冷的水喷出来，浇在他的旅游鞋上，浸湿了他的袜子。

### 4. 杨鹏《装在口袋里的爸爸》

**原文参照：**

> 我的爸爸确实只有拇指那么大，一天到晚坐在我的上衣口袋里，督促（这是妈妈的用词，其实是监视）着我的一举一动。
>
> ……他得通过椅子的扶手费尽九牛二虎之力爬到桌子上去，然后从电话机上像举起一辆汽车一样摘下听筒，跑到电话一端听对方在说什么，听完后急匆匆地跑到电话另一头，用他所能发出的最大声音同对方说话（不然人家听不见他的声音），一说完又以百米赛跑的速度跑到另一头去听对方的话，然后再跑回说话，再跑过去听话……打一次电话下来，他相当于跑了几千米的马拉松。

## 二、通用写作规则

以上四部原文共同遵守的规则，本书全文强制执行。

**① 句子是行动单位，不是描述单位**
每个句子做一件事：一个动作、一个感知、一句话。不要把三件事塞进一个长句。

**② 动词选择决定一切，不靠形容词**
"嗅嗅""踮起脚""跌跌撞撞地一闪"——这些动词带着画面，不需要形容词辅助。写完一段，把形容词逐一去掉，如果意思不变，就删。

**③ 感知写到物理末端**
不在感知停在"感到"的层面。"感到寒冷"→"冰冷的水浸湿了他的袜子"。感知要到达身体的具体部位，或者引发具体的行动。

**④ 内心活动是具体的念头或计划**
不命名情绪。"他感到紧张"→他在想什么、他身体在做什么。参照狐狸先生："嗯，他暗想，我看自己竟然还活着，这就够幸运的了。现在他们发现了我们的洞口，我们必须尽快搬家。"

**⑤ 荒诞用精确细节写**
越离谱的设定，越要用精确数字和平静口吻处理。参照《装在口袋里的爸爸》的逻辑链条，参照达尔的"他已经不停地吮吸了差不多一年了"。

**⑥ 段落控制读者呼吸**
紧张时极短，甚至一句话一段。放慢时才用稍长的段落。这不是排版问题，是节奏问题。

## 三、禁用句式

以下句式在上述四部参照作品中均不出现。

| 禁用 | 原因 | 替换方向 |
|------|------|----------|
| 取而代之的，是…… | 框架句，多此一举 | 直接写新的感知 |
| 眼前是一幅……的景象 | 框架句 | 直接描述 |
| 映入眼帘的是…… | 框架句 | 直接描述 |
| 以一种……的方式/语调/速度 | 从外部给动作打分 | 直接写动作 |
| 不是……不是……而是…… | 分析师思路，不是感知 | 直接写感知到的是什么 |
| 满满感到紧张/害怕/好奇 | 情绪命名 | 写身体反应或具体念头 |
| 心里涌起一阵…… | 情绪命名 | 写身体反应或具体念头 |
| 紧接着 / 与此同时 / 就在这时 | 人工过渡语 | 直接写下一个动作 |
| 氛围 / 透露出 / 弥漫着 / 俨然 / 不禁 | 成人书面词汇 | 换成10-13岁会用的词 |
| 浓郁的、甜腻的、混合着……的气味 | 形容词堆叠 | 一个精准词，或写身体反应 |

## 四、写完后的检查

1. **读出声来** ——卡壳的地方就是问题。
2. **搜"感到/觉得/心里"** ——每处都换成物理动作或具体念头。
3. **搜形容词** ——逐一问：去掉它，意思变了吗？没变就删。
4. **对照原文** ——反思这段话，符合中文儿童冒险小说的行文风格吗？
