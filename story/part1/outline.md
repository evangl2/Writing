# 故事大纲

## 使用说明

本文档是故事大纲，按章节记录已发生或计划发生的情节。

**记录方式**：白描式叙述，近似于电视剧前情提要。完整呈现情节走向和对后续有影响的细节——人物的决定、关键信息、伏笔、情节转折点。不做风格分析，不记录写作技法，只记事。

**使用方式**：

- 写新章节前，阅读已有大纲，了解当前情节状态
- 章节正文写完后，用实际内容更新对应大纲（以正文为准，大纲跟着正文走）
- 若章节包含占位内容尚未写入，在大纲中注明

# 第一章：满满的收集册

周四下午放学后，满满挤进晚高峰的地铁回家，心里复盘积分形势。今天晨会公布了"少年懂事王"竞赛积分，满满暂列第一，第二名马晓雨仅落后三分。这三分是马晓雨帮助老爷爷去急诊换来的，班主任马老师甚至打破打分系统，极其主观地破例加了十五点特殊道德分。满满感到强烈的危机感：不仅得防着无懈可击的马晓雨，在教室里还得随时提防那个活泼好动、上课画猪头捣乱的同桌苏乐带来的"纪律扣分炸弹"。

地铁上，满满翻看收集册里的奖项，回忆起上个月的朗诵比赛特等奖。比赛过程荒诞不经（评委形同虚设，故事是荒谬的母亲光脚踩玻璃三十公里买鞋），但妈妈却极其看重，精心拍摄九宫格照片发朋友圈进行充满虚荣和压迫感的“表演型育儿”打卡。这让满满对这些毫无逻辑的成规感到麻木。

收集册的最后一格留给"铁路独行侠"奖章——这是竞赛单项分值最高（十五分）的变态挑战：一天内完全无大人陪同，独自打卡十八条地铁线。班主任马老师宣布这项挑战后，班里号称"运动天才"的小网红周志远曾尝试挑战，结果因为没算准首班车发车时间而全盘崩溃，甚至因此掉粉。辅导班赵老师教过“大题切碎”的逻辑，满满以此为鉴，在脑子里制定了严密的路线和补给计划。

挑战定在明早（周五最后一次结算）。为了防备最大阻力——妈妈，满满计划明早五点在她熟睡时出门，留假纸条作伪装，搭乘05:13的首班车，利用到07:30早读前的137分钟"真空时间"完成连线打卡。当车厢播报"东方路站"时，满满准备下车，发狠认定马晓雨已经输了。

# 第二章：5:13 悖论

凌晨 04:45，满满起床。为了应对可能的变数，满满准备了被窝替身、解释纸条和回家挨揍三套方案。检查书包和装备时，摸到了同桌苏乐昨天硬塞进文具盒里的"纸青蛙"，这让满满紧缩的胃部稍微放松了一点。确认拉链全部润滑无声后，满满潜行穿过小区侧门，在 04:53 踏上前往地铁站的行程。一切都在精确控制中。

然而，在这个静谧的凌晨，意外开始浮现：原本熟悉的地铁入口被一堵凭空出现的实心水泥墙封死。满满被迫绕路前往 B 入口。B 入口的站大厅展现出一种极其不自然的尺度——天花板高得离谱，地面瓷砖映出的倒影仿佛通往深渊。满满在尝试过闸时遭遇卡片失效，在绝望地压在挡板上时，闸机却因某种乱码错误而开启。

在站台，满满遇到了一个穿深蓝色制服、两脚并拢站立、倒拎手电筒的守站大爷。大爷拦截了满满，但满满利用"数学题过保质期会变质发馊"的荒诞逻辑顺从了大爷的思维模式，成功获准通行。

05:12，首班车抵达。这是一列带着铁锈味和暗红色火花的银色列车，车门开启异常缓慢。满满在 05:13 准时登车，胃部刚开始放松，却在查看路线图时发现了恐怖的现实：05:13 是列车从起始站发车的时间，要抵达目前的站台至少需要十分钟。

这意味着，这趟在 05:13 准时出现的列车，根本不应该存在。满满陷入了"不在掌控之中"的虚无，车厢灯光诡异地闪烁。

# 第三章：检测到不规范行为

列车启动后疯狂飞驰，车厢屏幕的标准守则异化为血色警告，拉响了“检测到不规范行为：3/3”的防空警报并取消减速。生死关头，满满将生存危机当作期末考试大题拆解。在慌乱翻找测量工具时，文具盒里掉出了苏乐给的那只丑青蛙。最终满满通过“做题”逻辑成功破译并执行了指令：“切断所有客观测量”（强行按停正在计时的电子表）、“确保双脚接触现实”（双脚踩实列车地板）、“背负所有生存战略”（剧烈摇晃中套上沉重的双肩包）。警报解除，但列车已狂飙跳过多站，铁路打卡计划宣告破产。

满满按下紧急对讲机求助。对面换上了一个温和的男声，告知线路网络中根本没听说过满满每天出发的“东方路站”。满满崩溃大哭，男声安抚其两站后换乘，并应满满的执念，从对讲机里打出了一张荒诞的“迟到证明”（写明乘客已脱离运行时刻表，不属于任何钟表管辖）。临断线前，男声建议“艰难时记得给自己一个拥抱”。

为了寻找乘务员，满满试图拉开两节车厢的连接门，却发现门被焊死。愤怒踢打致使惩罚警报复燃，力竭且满腹委屈的满满只能顺着门滑坐，笨拙地紧紧抱住自己。随着呼吸平复、情绪着陆，死锁的门伴随一声脆响自动弹开。满满跌入一节新的车厢：散发旧书味、亮着钨丝灯的复古车厢，尽头站着一位穿着笔挺浅灰西装、戴着圆礼帽的绅士男人。满满走入新车厢之后，前一个车厢的屏幕显示出爱心符号与：“不规范行为：∞/0 ❤❤❤ 感谢您一直以来爱自己！”

# 第四章：箱子号与S博士

满满跌入的新车厢是一节极其高档精致的复古软座车厢。在这里，满满遇到了一位穿着西装、戴着圆礼帽的乘务员——S博士。S博士的工作台前布满复杂的机械装置，他通过一套复杂的物理连动机关给遵循“不喧哗”规则的满满分发糖果，并称这趟无名列车为“箱子号”。得知满满作为“异常变量”在首班车前就上车后，S博士如获至宝地记录起了实验数据。

满满向他说明了自己在前一节红车厢里遭受“惩罚”的经历，这让只信奉“正向奖励”的科学家S博士大为震惊与愤怒。他断定有人在他的实验系统内野蛮接入了错乱的惩罚程序。为了查明真相，两人拉开厢门，却发现红车厢已不复存在，取而代之的是一节正常的空软座车厢。为了重现规则，满满采取了极其夸张滑稽的违规行为（扔书包、劈叉悬空挂在行李架上、打开手表），但什么都没发生，甚至连手表的时间也彻底消失了。S博士平静地记录这一切，确定惩罚机制在此地已失效。

回到座位一通牛饮后，满满不再稀罕糖果和饮料，执意向S博士要求真正的奖励：一份能回家或去学校的全线线路图。然而S博士无情地点破真相：铁道系统过于庞杂，没有任何人拥有全卷线路图，想要抵达终点只能下车去站台里面"换乘"。面对因未知和孤身一人而陷入恐慌的满满，一直习惯用糖果作为解决手段的S博士，罕见地收起了按下奖励按钮的手，转而以抚摸手背的物理举动传递温情。他宣告了列车上一项不可更改的规则——"被抛性"（每个人都是孤身被抛入这趟列车），但安抚拥有极佳行动力的满满一定能平安抵达目的地。此时，列车冲破黑暗，带着温暖的光晕即将驶入第一站。

# 第五章：第一站

列车驶入一座巨大的古典候车大厅。半空悬吊的数百台老式电视机播放着色彩浓烈的直播与广告（番茄酱、火箭发射、蔚蓝海洋），其色彩与光束投射而下，把座椅上凭空出现的千百道虚无黑影“冲洗”成真实的实体乘客。然而，满座的乘客却陷入了绝对安逸的死循环中：有老太太不断拆掉围巾重织，有小孩反复擦掉作业重写。

面对外头拒不上车的乘客，S博士向满满承认，这座没有任何出口的站台正是由他亲手设计。他原意也是打造一座免却一切风吹雨打的完美温室，却终将这里变成了一座使乘客退化的舒适坟墓。出于极度自责与偏执的责任感，S博士拒绝在全员登车前独自发车，两人随着列车被死死困在站台。

# 第六章：推一把

为了打破死等僵局，S博士从地板下升起一套黑色金属控制台。他解释自己作为设计师已被系统锁死指纹权限，此刻只有“意外变量”满满才能操纵机器改变现状。为了能换乘回家、避开班主任马老师可怕的旷课惩罚，满满利用做选择题的排除法，选定了看起来最安全的绿色按钮“一步步不怕了”。

按钮启动了长椅的阶梯性加热功能。S博士将此视为科学温和的脱敏训练，并讲了一个“恐高小孩通过不断垫高书本克服恐惧去爬雪山”的案例。在逐渐加温下，大批受不了滚烫座椅的乘客终于落荒逃入车厢，大厅只剩最后一名反复浇水硬抗高温的顽固老兵。此时S博士抛出“半途而废”四个字，瞬间引爆了满满对学校与家庭惩罚的条件反射式恐惧。在极度紧张的创伤应激下，满满砸下了极限加热。老兵在高温中自燃烧起，却在烈焰中挺拔站直向列车敬礼。列车终于发车启动，火海吞没了大厅，暴露出真正的站牌“怀旧站”。列车一头扎进幽暗的隧道，满满凝视车窗倒影里的脸庞，却认不出那是谁。
