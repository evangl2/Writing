# 文件档案清单 (File Inventory)

## 1. 文档作用与记录方式

本文件定期记录项目核心目录下所有文件的基本属性，作为创作过程的物理存根。它能帮助我们追踪各章节的更新频率、篇幅变化以及设定文档的演进规律。

**记录条目**：
- **名称与路径**：基于项目根目录 `Writing/` 的相对路径。只记录story和setting和traking文件夹下的所有文件。
- **时间戳**：采用 `YYYY-MM-DD HH:MM:SS` 格式（当地时间）。
- **文件体积**：以字节（Bytes）为单位，推算成字数。
- **状态**：用NEW标记所有最后修改时间产生变化的文件。删除没有变化的NEW标记。
